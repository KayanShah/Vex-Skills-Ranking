## вңҰ What is this?

A fast, fully client-side scouting web app for the **2026 VEX Robotics World Championship** (Middle School). It lets you instantly preview and download beautifully formatted **skills standings PDFs** for any of the 7 competition divisions вҖ” or the entire Worlds field at once.

No backend. No database. No login. Just a single HTML file that runs entirely in the browser.

---

## вңҰ Features

| | Feature |
|---|---|
| рҹҢҚ | **All Worlds ranking** вҖ” every team across all 7 divisions in one ranked list |
| рҹ“җ | **Per-division rankings** вҖ” Arts, Engineering, Innovate, Math, Science, Spirit, Technology |
| рҹ“„ | **PDF export** вҖ” beautifully formatted with HABS Gliders branding, medal highlights, and clickable hyperlinks |
| рҹҘҮ | **Medal rows** вҖ” gold / silver / bronze highlighting for top 3 in each division |
| вҡЎ | **Instant preview** вҖ” live ranked table in-browser before you download |

| рҹҡҖ | **Auto-deploys** вҖ” push to GitHub вҶ’ Vercel rebuilds in ~30 seconds |

## вңҰ PDF Output

Each PDF includes:

- **Dark navy banner** with HABS Gliders logos on both sides, event title, division name, and team count
- **Full ranked table** with Div #, Global #, Score, Auto, Driver, Team #, Team Name, Country
- **Medal row highlights** вҖ” gold / silver / bronze for the top 3
- **Per-page footer** on every page: page number В· division label В· *[Kayan Shah](https://github.com/KayanShah)* (clickable hyperlink) В· HABS Gliders Scouting
- **End block** вҖ” centred logo, team name, clickable author credit
- **No-score section** listing any teams with no skills data
- **Timestamp footer** вҖ” data freshness note + ranking methodology

The **All Worlds PDF** generates in **landscape A4** with an extra Division column.

## вңҰ Updating the Data

Skills data is embedded in `index.html` as a JS constant. To update:

### 1. Update skills scores

Replace the `skills` object inside the `const RAW = {...}` block with fresh data from the VEX skills standings CSV.

### 2. Update division team lists

Replace the `teams` arrays inside `RAW.divisions` with the latest official team lists from RobotEvents.

### 3. Deploy

```bash
cd vex-skills-app
git add -A
git commit -m "Update skills data вҖ” [date]"
git push
```

Vercel auto-deploys in ~30 seconds. Works from private repos too.

## вңҰ Ranking Logic

```
1. Filter division teams against global skills standings
2. Sort by Score DESC
3. Tiebreak by Global Rank ASC (better global rank wins)
4. Assign division rank (#1, #2, вҖҰ)
5. Teams with no score listed separately (unranked)
```
